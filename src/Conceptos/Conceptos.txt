Conceptos Importantes:

Component (componentes): Son como tipo secciones que retornaran un 
elemento HTML.

Props: son propiedades asi como parámetros para un componente tipo
function

State (Estado):

Hooks (ganchos): nos permite tener estados en una function ya que antes
obligaba a crear componente con class para acceder a todas
las posibilidades de la librería

Event Listener: 

JSX: Es una extensión de react para la sintaxis de JavaScript permitiendo
escribir componentes con una estructura similar a HTML, en pocas palabras es
JavaScript y HTML juntos.
